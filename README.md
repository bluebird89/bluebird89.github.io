# Knowledge

个人记录

## 原则

* 效率
* 体系构建
    - 看书
    - 做思维导图
* 舍弃
* 反思
* 知行合一
* 学习》加班

## 错误

* 知识是一个风口，很多转载，以及内容参差不齐
* 贪多，不求甚解。看过东西想清楚、能讲清楚
* 不同网络、不同搜索引擎，搜索结果不一样
* 博客问题:知识点比较散

## 习惯

* 周期时间内的克制
* 开始写 todo list，理清 focus on what’s important；结束写反思。应用到每天、月、季以及每一件事
* 每月读一本书或一个框架
