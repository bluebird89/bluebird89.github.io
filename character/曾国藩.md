# 曾国藩

* 曾玉屏是湖南长沙府湘乡一个普通农民，早年是个游手好闲的精神小伙。突然一天听到邻居教育孩子：莫学曾家儿子整日游手好闲，曾家早晚败在他手。曾玉屏听后大受震动，决心痛改前非，脚踏实地做人。从此他起早贪黑下地干活，几年过去便有了几亩良田，还雇了长工。
* 邻人占了他家几分田地，曾玉屏大字不识，便托秀才帮忙起草诉状，将那人告上衙门。岂知那秀才早被恶邻收买，颠倒黑白，衙门老爷葫芦僧判断葫芦案，曾玉屏因不识字吃了哑巴亏
* 曾国藩五岁开蒙，八岁读完五经，九岁学做八股，不可谓不用功。曾玉屏把没能在儿子身上实现的梦想全部转移到长孙身上。身教更胜言传，祖父的坚忍持重的品格也潜移默化地影响着曾国藩的一生
* 前前后后六次考秀才接连失利。但残酷的现实也让曾国藩认清了自己。他绝不是聪明的小孩
* 道光十三年（公元 1833 年），22 岁的曾国藩终于考中了秀才。而比曾国藩小三岁的洪秀全，作为全村的希望四次落榜，没能出道，走上了一条未曾设想过的道路
* 举人和进士却考得相当顺利。道光十九年（公元 1839 年），28 岁的曾国藩被任命为翰林院检讨，从七品，相当于副处级
* 年底，道光帝命礼部侍郎曾国藩考核顺天府县学.先把满族秀才们平时的功课悉数检查一遍。文章大多不堪入目，啼笑皆非。又找来县学教授问话。质问他如何教的这些“好学生”。那满族教授却说，我大清江山靠的是马背上的功夫，而不是文章！曾国藩不动声色，连夜奏明道光，革除了这人功名.教授指着曾国藩大骂，你不过是满人的一条狗！曾国藩回怼，我不仅是满人的狗，还是汉人的狗，我是大清国的一条好狗！
* 当夜，十几个满族秀才手持棍棒闯入曾国藩行辕，想要教训教训这个不识时务的汉官。曾国藩赶忙翻墙狼狈逃出。他选择了汉军驻扎的绿营，而没去满军的旗营.曾国藩带领汉军立刻杀回，把闹事之人悉数抓获，严加看管。次日便向道光帝汇报此事。道光帝原想革除这些秀才的功名敷衍了事，但曾国藩冒死力奏，必须处死这些暴徒，以正视听！午时三刻，随着三声炮响，十九颗人头同时落地，“曾剃头” 名不虚传
* 咸丰二年（公元 1852 年），四十二岁的曾国藩回老家守母丧，守孝不足百天，朝廷就下达紧急命令，让他在湖南帮办团练，抵抗一路横冲直闯的太平大军。从此曾国藩开启了书生带兵的传奇生涯。
  - 招兵对象主要是湖南农民，一来同乡便于管理，二来觉得湖南民风彪悍，骁勇善战.遴选的湘军领导团队也基本都是湖南本地知识分子
  - 清朝的正规军军纪废弛，还看不上湘军这些土包子，双方经常发生争执械斗。某次，满将鲍起豹领着清兵闹事，把曾国藩的官邸围了起来，眼看军中失控。前来公断的巡抚只是简单训斥了几句了事，巡抚大人旁若无人地说道，以后打仗还得指望他们呢，说完拂袖而去。带领湘军进行战略转移，离开长沙，去往僻静的衡阳操练。与考察县学被围攻事件不同，这次采取了截然相反的态度。选择不逞一时之快，退步忍让，努力创造有利局面，迅速壮大湘军，以备他日之用
  - 到衡阳不久，好友郭嵩焘向他建议，要建立水军以守长江之险。曾国藩采纳建议，成立衡阳造船厂，遍招能工巧匠，大量建造战船。当时全国造炮工艺落后不堪重用，曾国藩又从广州大量购置洋炮装在战船之上。通过不懈努力，水军成为当时中国最先进的内河水师
  - 湖北、安徽战事接连吃紧，咸丰屡次三番下旨命曾国藩出战迎敌。但曾国藩坚持以大局为重，以湘军没做好万全准备为由多次抗旨。将在外君命有所不受，焦头烂额的咸丰帝拿曾国藩没有办法，再说正是用人之时，也不好发作，只得鼓励曾国藩好好练兵。太平军士气正盛，自己若轻举妄动、冒然出击，可能全盘皆输。
  - 咸丰三年（公元 1853 年）十二月十七日，初冬的寒风刚刚扫净街上的落叶，孤立无援的安徽巡抚江忠源战死沙场。上到清廷，下到黎民，皆对曾国藩 “缩头乌龟” 的做法嗤之以鼻，曾国藩却不气恼
  - 咸丰四年（公元 1854 年），由于情报有误，湘军中了埋伏，先失一阵，曾国藩的首战失败，羞愤的差点跳河。但是湘军毕竟训练有素，再战，取得湘潭大捷，终于拿下长沙。曾国藩又一鼓作气收复了武汉。
  - 再战江西时，因战前准备不足又掉入了太平军的陷阱，差点被俘。困守江西之时，曾国藩父亲去世，曾国藩请假三月回家治丧。咸丰早就厌烦了这个不听话的湘军指挥官，马上令他守孝三年，这等于夺了曾国藩的兵权
  - 咸丰十年（1860），曾国藩加兵部尚书衔，署理两江总督，管辖江苏、江西和安徽三省（明朝江苏和安徽合称江南省，故称两江总督）
  - 同治三年（1864），湘军挖的一条地道成功穿过南京城下。随着几万斤炸药的巨响，城墙被崩了一个大口子，湘军蜂拥而入，太平天国运动被剿灭
  - 主动裁撤湘军，还逼战功赫赫的弟弟曾国荃辞职返乡
* 同治七年（1868），曾国藩由两江总督调任直隶总督。直隶省包括天津河北以及河南、山东小部，总督衙门设在保定
* 同治十一年（1872）三月12日下午，突发中风，不久便与世长辞。朝廷追赠太傅，谥号“文正”，挽联这样写道：立德立功立言三不朽，为师为将为相一完人。
