
## Go##

### Install ###
- download:

	`wget https://storage.googleapis.com/golang/go1.8.linux-amd64.tar.gz` 
    `sudo tar -C /usr/local -xzf go1.4.2.linux-amd64.tar.gz`
    
### Environment variable ###
    export PATH=$PATH:/usr/local/go/bin

### Build&Run ###
    go build hello.go
    ./hello
