        + 一直到了最近几年，为了继续向前走，才开始注意到，考虑面向大众，因为音乐是需要被更多人听到理解，才会被喜欢的。
    - 最早是一支以“金属车间的形体师傅”命名的Punk乐队，由彭磊、尚笑、刘葆三位北京工艺美术学校的学生构成。1997年，彭磊就和尚笑、刘葆在距离工艺美校三个路口之外的北京服装学院防空洞里排练了几天后，去找师兄沈黎晖录了歌，他们给自己改了个通俗易懂的名字“新裤子”。这首歌就收录在《摩登天空1》合辑打头阵的A面第一首歌里，这首歌就是《我们的时代》。这张专辑的封面设计师是庞宽，“This’s our Times”作为歌词印上了《摩登天空1》的封面，不经意间就开启了与摩登天空和 “北京新声” 的故事。
    - 彭磊接触到摇滚乐。当时唐朝发行的磁带太火了，彭磊拉着朋友们在一起练习重金属，但是他的队友技术都太弱了根本搞不了。有一段时间他们特别迷茫，当时很多人开始喜欢涅槃（Nirvana）了，就是那种主歌一直低音唱，到副歌忽然翻八度开始大声嘶吼的那种。
        + 当时许巍、汪峰都开始这么搞而且火了，电视节目里也特别多人这么唱，变成了一个特别可怕的套路。
        + 但彭磊他们当年不是特别喜欢涅槃（Nirvana），觉得这样唱有点儿傻。而许巍、汪峰的歌都太苦了，根本不是他们生活里所有的喜欢的东西。
        + 彭磊误打误撞买了一盘雷蒙斯（Ramones）的磁带——和弦简单，劲又很足，大家一致认为“这才是我们想做的音乐！”这之后新裤子乐队就转向了朋克。 后来彭磊回忆彼时的经历：“我们当时什么都没见过，看什么都觉得新鲜，但紧随而来的又是厌倦。”
    - 1999年新裤子录制了第二张专辑《Disco Girl》，庞宽以机器人的身份，带着合成器正式加入了乐队。虽然还是朋克的底子，却多了些都市的纸碎金迷和年轻人的享乐主义。
        + 在《流行一代》里，大舌头的彭磊第一句就唱道：“我们都已改变，理想还没实现”。朋克男孩们开始了音乐上的第一次主动背叛，而关于“改变”的争议贯穿了新裤子发展的每一个节点。
    - 2006年，新裤子乐队的《龙虎人丹》彻底变成了新浪潮和合成器流行，结合经典国货、港式迪斯科舞厅、社会主义初期的复古设计、国产武术、胡同洗头房的美学符号，他们掀起了不小的经复古时髦的潮流。
        + 极尽夸张的视觉造型和舞台呈现引起了不小的狂欢，但也导致了乐队创始成员刘葆离开，随后加入了老牌朋克乐队蜜三刀。
    - 2012年，拜合成器和复古Disco所赐，新裤子与80年代的Disco女皇张蔷合体了，然而现场音乐的主要受众——90后观众对这样的跨时空组合毫无接受准备，只是觉得时髦又有距离感。
    - 2013年创作出了《别再问我什么是迪斯科》，从生活的琐碎到音乐上的表达无一不在为那个他们所熟知并喜爱的年代释放深情。在甜蜜的创作期过后，新裤子与张蔷一起在音乐节上演出，慢慢又变成了一种职业乐队的状态。
    - 2015年新裤子发行了单曲《关于失眠和夜晚的世界》，这首歌释放了一个信号，那几个潇洒、嘲讽、无所谓的北京少年，开始识得愁滋味，开始突然像在北漂一样唏嘘感慨生活的不易和艰辛。
    - 几年之后，他们开始回归三大件的摇滚乐，从《没有理想的人不伤心》之后，他们宣布走进“黑暗时代”，重新穿上皮夹克。
    - 分别是彭磊、庞宽、赵梦、Hayato，音乐上也回归传统的摇滚乐
    - 彭磊
        + “科班出身”，毕业于北京电影学院动画专业。和他同届的，还有黄晓明和赵薇
        + 动画片《可可可心一家人》的导演，甚至在2006年执导了神曲《QQ爱》的MV
        + 没有底气再啃老，同时又得养小，以及发现自己可能永远实现不了财务自由。他们已经失去了很多希望和幻想，但还没有完全接受现实
        + 曾经形容自己有反大众人格，觉得别人喜欢的都是破玩意，他讨厌媒体，讨厌宣传，去演出时看到人很多也会觉得烦躁
        + 2010年，彭磊把自己卖不出去的画集合成册，做了一本小书《北海怪兽》，更像是关于新裤子的一部传记，在这本书里彭磊以漫画的形式第一次讲述了自己的成长故事，那些看似潇洒玩乐无所谓的态度背后，隐藏着很多生活的无奈和心酸。
        + 彭磊受父亲的影响，从小去少年宫学画，然后去上了当时的中专“北京工艺美校”。如果没有做乐队以及被同门师兄沈黎晖发现挖掘，他就是去画脸盆、痰盂的工人。
        + 《乐队》讲的是新一代摇滚青年的现实状况，也正是因为这部电影，彭磊获得了上海电影节亚洲新人奖最佳导演。
        + 说实在的，因为大家到现在还是特别平凡，当然比90年代心态好了，但是大家都老了。
    - 庞宽和彭磊一样在他们心中或多或少都住着那个文艺青年，但不同于彭磊的艺术形式，他更夸张、冷峻和神秘。
        + 在新裤子的第二张专辑《Disco Girl》时，以机器人的身份加入了乐队，对于他来说，机器人是一个人类幻想的延伸，无论技术怎样进步，却似乎永远无法满足制造者的欲望。
* 刺猬乐队
    - 子健:北航毕业 高中时代听Nirvana开始，他也就认定自己喜欢这样的音乐，要做出比这更牛逼的音乐
        + 都没有想过太多明天，信奉“与其苟延残喘，不如从容燃烧”
        + 可是慢慢世道变了，体面地活着变成了更艰难的事。
        + 今年36岁，在他身上半点找不到“成熟”的踪迹。与同龄人相比，他似乎一直有所滞后
        + 一个社会化比较低的人格
        + 最难受的是觉得不是没有做过努力，而是做了各种尝试想融入集体：去上班、和同事相处，最后发现还是没融进去，还是没有几个能走心的人。
    - 石璐
        + 2015年，Nova Heart乐队在愚公移山的一场演出中，石璐兴奋地向观众宣布自己“有喜了”！
        + 从一个暑假在少年宫里打鼓的小姑娘，到一个摇滚乐队的女鼓手，再到女鼓手与男主唱恋爱分手长大，与别人结婚生子离婚成为单身妈妈。
        + 子健总是有一个特别强的时间感，他觉得此时此刻，现在一年之内我要表达的就是这样，不用它多完美。但是如果再等再慎，这东西就没了，他也不想唱了。
    - 事迹
        + 赵子健和石璐从刺猬乐队组建第二年成了一对情侣。2008年的鼓楼，赵子健与石璐租住的平房不到15平米，只有一个房间，夏天潮湿冬天漏风。
        + 有一次两人从美国巡演回来，发现平房的玻璃被路过的挖土机敲碎了，玻璃碴和垃圾洒了一床。但那并不是赵子健住过最糟糕的地方，他还经历过更糟的。
        + 曾经有阵子他房租到期又辞了职，没有收入，更不想回父母家。赵子健就索性在他们长期租用的安德路的那间排练室支了个床，天天住在里面。排练室在地下二层，潮湿，阴暗，全是螨虫。有时早上醒来，子健发现整个脸都是肿的。
        + 赵子健和石璐的恋爱终结于2013年，那阵子他们依然生活穷苦，日子过得更乱七八糟了。除了经济压力、状态不好，在社会中一直处于边缘化，赵健更不怎么爱与人相处了
        + 2010年，刺猬的第一任贝斯手朱博譞与赵子健在音乐理念和审美上的分歧越来越大，最终朱博譞选择了退出，何一帆在这时候加入刺猬。石璐的考量是，他是个“未来也能稳得住的人”。
        + 刺猬有一种不断拖着自己的青春，但最终只能看它走远又无能为力的悲壮状态。
