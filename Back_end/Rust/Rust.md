# Rust

## 安装

```sh
curl https://sh.rustup.rs -sSf | sh

brew install rustup
rustc --version
```

## 配置

所有工具都安装到 ~/.cargo/bin 目录， 并且您能够在这里找到 Rust 工具链，包括 rustc、cargo 及 rustup。

* [Wilfred/remacs](https://github.com/Wilfred/remacs):Rust heart Emacs

## 管理

通过rustup管理
