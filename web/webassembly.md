# WebAssembly

* 一种适用于 Web 的可移植编译格式，提供更小的文件尺寸和更快的加载速度
* WebAssembly 模块以二进制的格式发送到浏览器，并在专有虚拟机上执行
* 这个虚拟机与 JavaScript 虚拟机共享资源，如内存和线程。WebAssembly 模块总是与 JavaScript 代码一起使用，在必要的时候可以执行一些有用的操作。
