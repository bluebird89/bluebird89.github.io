## 台湾

* 1949年，随着解放战争的胜利，蒋介石败退到了台湾.随着蒋介石来台的，不止是民国的学界耆宿，政商要员，还有数不清的国民党底层官兵。
    - 为了安置这些跟随蒋介石戎马半身的官兵，台湾当局特意批了郊区的闲置土地，兴建了眷村（相当于部队大院，不过层次很低）
    - 眷村在台湾的社会是特殊的存在，老一辈的不是台湾人，一心想回大陆。
* 立足台湾蒋经国讨好本省民众,谋求大权李登辉联合党外力量
    - 从反攻大陆无望之后，国民党便收了心思专心经营台湾。一方面制订出口为导向的国策，另一方面扶持本省派精英，希望自己在台湾的统治愈发稳固,引来了国民党内外省派的不满
    - 蒋经国为了安抚外省派，只得授予本省派一些虚衔，比如李登辉就被扶上了副领导人的位置
    - 1988年蒋经国突然过世，接班人尚未选定，李登辉就任代领导人。党内本省派的力量顿时有了靠山，开始准备与保守派争夺党内大权
    - 为了削弱外省派势力，打着民主自由的旗号，开放党禁、地方选举。一时间受国民党压迫已久的党外人士纷纷站了出来，进行组党
    - 1990年，国民党本省派和外省派的争执达到了白热化，外省派准备趁着领导人选举的机会，换掉李登辉，打击本省派。而李登辉也在暗地里联系民进党，叫他们组织游行，打击国民党统治
    - 民进党成立后，早就想谋求台湾的统治权，于是他们欣然同李登辉结成了统一战线，组织了野百合学运，包围了阳明山（台湾当时立法会所在地）。 声势之大，让保守派胆战心惊。李登辉趁势提出谈判，他劝说党内的元老，说民主改革既是总理遗志，也是大势所趋。这样闹下去，怕是要酿成流血事件，到时不好收场
    - 党内元老同意李登辉继续担任领导人，叫他带领本省派同民进党谈判,同队友谈判可能是这个世界上最轻松的工作，李登辉很快和民进党达成了一致：4年内开放台湾领导人直选。就这样，国民党得了一个可能继续执政的面子，民进党得了或许能执政的里子。袁世凯的套路
    - 国民党内的外省派不甘心，担心本省派会与台独势力暧昧不清，一部分党员选择出走另立新党
* 维护道统眷村太保参与竞选， 守护中国本省元老退党自清
    - 本省派本就和台独势力暧昧不清，而且得势之后还经常提出削减老兵退休金的提案，这让眷村出身的韩国瑜不参选台北县议员
    -　遭受打击的外省派，也急需新鲜血液的补充，以恢复力量。他们全力支持韩国瑜这种眷村出身的小辈，为他们拉票。最终韩国瑜顺利当选台北县议员
    -　民进党，自从开放选举之后，凭借在本省人中深厚的群众基础，在地方选举中每每获胜，风头一时无二。但就在高雄市议员选举前夕，创党委员许崑源却宣布退党参选，舆论哗然
        +　一开始只是为了推行民主化，反对国民党专治，才选择组党，鼓吹台独显然不是他们的初衷
        +　从得势之后，民进党就抛弃了要民主，不要台独的主张。为了讨好选民鼓吹独立，宣扬一边一国。这让很多创始党员不满
    -　许崑源
        +　出身高雄，是个地道的本省人。但是从小家风甚严，父亲在日据时期还曾参加过反日组织，在当地颇有名望。在家庭熏陶之下，许崑源打心里觉得自己是中国人。而这次民进党鼓吹台独，让性格直爽的他感到愤怒
        +　大肆抨击民进党分裂国家，将台湾带入深渊，宣布不与台独分子同流合污，奋而退党
        +　退党的新闻在全台引起轰动，韩国瑜看着电视对妻子笑着说：“没想到民进党也不全是台独。”
        +　2008年加入了国民党。却发现国民党为了自己的民调，总是对两岸关系含糊其辞，能拖就拖，让他不满，他主动请缨去竞选立法委员
* 1980年，台湾人民不满国民党政治高压，两万人在《美丽岛》杂志社附近集结游行，为争取民主示威。游行最终演变成警民对峙，台湾当局出动了镇暴部队。
* 1980年2月28日，游行的发起人林义雄遭到政治报复，他的家人全部被杀，只有长女重伤幸存。女孩躺上手术台，那天，医院里当班的正是罗大佑。手术结束，罗大佑留下了女孩的胸腔穿刺针。
* 那时他刚写完《光阴的故事》，这次事件之后，罗大佑把眼光投向了台湾社会。做实习医生期间，罗大佑花光积蓄做了一张专辑。他找到滚石，决定出完专辑就遵从父亲意愿，做一名医生。 那张专辑是《之乎者也》，轰动了整个台湾。第二张专辑《未来的主人翁》，批判精神更加明显，之后，罗大佑脱下了白大褂，成为了歌手，却被称为台湾社会的手术刀。连台湾音乐界的大哥李宗盛都说：和他（罗大佑）相比我会感到惶恐。
* 高晓松说：“罗大佑和崔健一样，都是不可逾越的，是他俩让我们明白了，流行音乐不仅可以谈小情小爱，还可以表达讥讽和愤怒，为人类文明的堕落拉响警报。这俩人把华语音乐提到了一个崭新的高度。”
* 台湾实行民主化改革之后，立法院涌进了大量民进党人士。但国民党的思路，一时间还没从两蒋时代转换过来，质询时总是慢慢吞吞，看着发言稿照着读.民进党从街头政治兴起，深谙诡辩之道，他们在立法会，大声恫吓，携带道具对国民党冷嘲热讽，甚至于敢于直接上手，在立法会里上演全武行
* 1993年5月5日，立法会正在讨论关于削减老兵退休金的提案。 代表民进党发言的正是日后成为台湾地区领导人的陈水扁。 他眯着，挂着他标志性的笑容，说荣胞（来台老兵）中心的预算过高，那么多的钱给那么少的人，是一种资源浪费，难道我们要把荣胞当猪养吗？此言一出全场哗然，此时韩国瑜站了起来走到陈水扁面前，上去就是一个耳光。陈水扁应声倒地，保安一拥而上现场乱做一团
    - 经过民进党的挑拨，民众已经把对国民党的恨扩大成对老兵的恨。认为他们帮助国民党欺压本省人，抢走他们的工作机会。而现在我们台湾人交的税居然还要拨给他们当养老金，这让民众纷纷站在了陈水扁这边
    - 被打之后，陈水扁适时地住进了医院，说自己轻微脑震荡，可能留下后遗症。此举又搏得了民众的同情
* 1996年，新的“敌人”似乎出现了。3月8日，两枚东风15导弹从福建永安点火升空，迅速掠过了台湾海峡，飞向了基隆与高雄外海，作为对李登辉访美的回应
* 克林顿政府行动迅速，立即调动“尼米兹号”和“独立号”两个超级航母战斗群，把它们部署到了台湾附近海域
* 剑拔弩张的96台海危机，给大陆和台湾留下了数不清的野史和传说，但对于太平洋彼岸的美国来说，这场危机让他们重新审视了这个80年代蜜月期的伙伴：拥有300万常规部队，GDP全球排名第8，发展道路又跟西方不同。在灯火阑珊处，苏联腾出来5年的位置，似乎有了新的人选
* 2000年大选，大选结果揭晓。国民党失败，被迫让出把持了半个世纪的权力。此时的许崑源在高雄，和国民党支部取得联系，表示自己有意加入国民党，维护道统
* 韩国瑜
    - 1957年，在眷村里，韩济华迎来了他第六个孩子，并给他取名韩国瑜.像韩国瑜这样的眷村二代，则在大陆人还是台湾人的认识中苦苦挣扎
    - 在眷村生活的时候，韩国瑜的父母只会买便宜的竹子家具，韩国瑜不解，父母回答他：“买那么贵的干什么？过两年就要打回大陆了。”
    - 学校的台湾同学可不理会这么多，他们看韩国瑜这些从眷村出来的小孩不会说台湾话，便经常上前找茬。被欺负多了，韩国瑜自然也就习惯用拳头回应。不仅自己打，还经常帮眷村小孩出头。渐渐地在学校里他成了学校里眷村帮的老大，一个货真价实的小混混
    - 直到有一天，他回到家，见家中添置了很多贵重的木制家具。而父亲在一旁默默喝酒，母亲则在含泪收拾卫生。家中没有人说话，气氛十分凝重。这时韩国瑜知道，大陆回不去了
    - 1975年他考上了军校，退伍之后，考上了东吴大学，而后更近一步考上台湾政治大学的研究生,在研究生期间，他应聘上了台北市议员的助理半工半读，赚到不止是一点微博的生活费，还结识了前来采访的女记者——李佳芬，两个人后来喜结连理，成为夫妻
    - 毕业后加入了国民党，想着继续为蒋经国三民主义，统一中国的理想奋斗
    - 1990年，野百合学运成功，国民党党内大权旁落。 当时还是记者的韩国瑜怒不可遏。得势之后还经常提出削减老兵退休金的提案，这让眷村出身的韩国瑜不能接受。维护国民党的道统，他站了出来，准备自己参选台北县议员。没有背景的他胜选机会渺茫，为此他找来了自己的妻子——李佳芬进行商议
    - 李佳芬
        + 出身云林，李家在当地是有名的大家族，她的父亲更是两蒋时代的县议员，政治背景雄厚
        + 一开始深谙政治险恶的李佳芬并不同意，但后来韩国瑜一连苦求数日，为了满足丈夫这个梦想，李佳芬最终松口，动用自己家族的力量为韩国瑜造势
    * 1992年5月21日。台北县一家保龄球馆突发大火，造成20多人死亡。当时担任台北县县长的是民进党籍人士——尤清。他为了支持民进党的选举，派警察和消防去拉票现场维护治安，罔顾火场灾情。
    - 在县议会里韩国瑜就此案，要尤清解释清楚。尤清却说，如果犯法的话，让法律定夺。我觉得在法院没有给出结果前，我不应该受你的指责。国瑜来了火气，直接呛声：“出了这么大的事情，工商局，建设局，警察局，居然一个都不用负责，县长，你好笑哎。”尤清不置可否，搪塞过去。这让韩国瑜更加恼火直接大声说道：“国之将亡，必出妖孽。尤清，出的就是你这种妖孽！”
    - 韩国瑜一战成名之后，引起国民党高层的注意，派人向他询问竞选立法委员的意愿
    - 党内资源的倾斜加上质询尤清时所获得的人气，韩国瑜以巨大优势赢得竞选。顺利当选立法委员的韩国瑜意气风发
    - 国民党力挺他竞选立法委员，除了对抗民进党的需要之外，还有一个最重要的考量——他是个没背景的外省派
    - 一次民进党上前阻挠法案表决，韩国瑜抄起桌上的水杯劈头砸了过去，民进党还没反应过来，韩国瑜就已飞起一脚，将阻挠表决的议员踢翻在地
    - 以党主席李登辉为首的本省派，为了迎合选民，已经抛出一边一国论调，阻止台商赴陆投资，离间两案关系
    - 不止一次在立法会中提出，国民党内某些人失去中心思想，背叛中山先生遗志。这让党内元老觉得难堪，而后又与之前出走的国民党外省派联系紧密，想劝说他们“回家”，共同匡扶国民党的道统
    - 1993年，党内也终于抓住了韩国瑜的小辫子，曾经火爆浪子遭遇危机
    - 韩国瑜是眷村出身，此时陈水扁侮辱老兵，让他想起了儿时坚持不换家具的父母。他们远离故乡，漂泊到这个小岛，为台湾的发展尽了自己的本分，到老了还要被你们这样侮辱。如果不还击，简直枉为人子。
    - 恰巧国民党高层也对韩国瑜的言行不满，准备借这个机会敲打敲打他，于是党内不仅不为韩国瑜辩护，反而逼他道歉.在压力之下，韩国瑜召开新闻发布会，宣布对自己的暴力行为道歉,“但陈水扁侮辱荣民，是不是也要道歉？”韩国瑜对着媒体说道.从此韩国瑜被国民党冷藏，一直不能再有作为
    - 淡出政坛的韩国瑜，回到了妻子的老家云林县，在那里创办了双语学校，过着乡绅的生活倒也怡然自得。除了管理学校，还有就是帮农民卖菜
    - 2008年时，因为气候原因，导致云林县蔬菜滞销，农民叫苦不迭。首先以学校的名义买进滞销蔬菜，充作学生午餐，而后还主动前往台北向农业部门主管的台北农产运销公司（简称北农）进行推销
    - 在推销过程中，他认识了农会总干事——张永成。这张永成也是出身云林，一看老乡有难，很自然地答应了韩国瑜的请求。他对韩国瑜的义气之举，十分赞赏，在谈完生意后，便留他喝酒。一来二去，两人成了好友
    - 2012年，正是这位好朋友的托付，让他有了重返政坛的机会，当时北农业绩下滑年年亏损，而菜价却不断高涨，让百姓叫苦不迭。撤换北农总经理成为当务之急。但情急之下却没有合适人选，这让张永成头痛。想请韩国瑜出任总经理，一向重义气的韩国瑜立马答应。2013年1月1日，韩国瑜回到台北，正式就任
    - 发动薪水改革，将公司所得利润分成给旗下员工，员工一看人人有钱赚，工作也就分外卖力，也正好恰逢台湾遭遇台风，菜价上升，北农居然转亏为盈。张永成看着业绩报告，便抓着韩国瑜不放，韩国瑜无奈只得继续留任
    - 总经理的职位让民进党眼馋。民进党立委攻击他有流氓性格，有打人的历史，不适合担任。一开始并没有在意，反正这也不是第一次了。但是后面开始污蔑贪污公司财产，并且质疑发放员工奖金，属于浪费资源
    - 2016年10月25日，韩国瑜在办公室召开记者会，办公室没有进行布置，只是桌上摆了一盘曲棍球。直斥，民进党立委是不是书读到狗肚子里去，满口胡言。 “在我男人的世界，你们这种行为就叫小瘪三，听好了民进党你们是小、瘪、三！”说着韩国瑜拿起一颗曲棍球说，“我的账目都在这里，如果我贪污，我就把这一盘球都吃下去。如果我没贪污，你们不用吃一盘，吃一颗就好！”
    - 11月15日，台北立法会，民进党籍议员王世坚对韩国瑜发动了攻势。说他是流氓当道，黑道治国，韩国瑜不甘示弱，如果我没有贪污，你是不是去跳海？你在这里左一个流氓，右一个流氓，是什么意思？两人大战立法会的视频被传到网上，引发网民狂欢，国民党的支持者发现，原来党内还有这等硬汉
    - 一夜爆红使韩国瑜始料未及，同样没有想到的还有民进党。他们一见韩国瑜的人气急速蹿升便把矛头指向了张永成。说他管辖无方，手下不仅有黑道历史，还在立法会哗众取宠。一时间关于张永成的指责铺天盖地。考虑到朋友的处境。2017年1月11日，韩国瑜请辞北农总经理。辞职当夜，员工们出来挽留，张永成也打电话出来劝他不要担心自己。摆了摆手，自己去意已决，有空出来喝酒，挽留之事不必再提。 张永成了解朋友性格，也就不再多说。
    - 2017年5月20日，竞选结果出炉，韩国瑜得票率仅有4%，竞选失面对媒体说出了自己的想法：“国民党党章里写了，大陆与台湾是一国两区，而现在却有人讲台湾是主权国家这是叛党行为。而且党内的老人长期把持权柄，让年轻的才俊无法出头，这样下去，国民党将会烂成什么样？”败。面对媒体说出了自己的想法：“国民党党章里写了，大陆与台湾是一国两区，而现在却有人讲台湾是主权国家这是叛党行为。而且党内的老人长期把持权柄，让年轻的才俊无法出头，这样下去，国民党将会烂成什么样？”
