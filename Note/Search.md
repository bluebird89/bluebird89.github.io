# 搜索


## 技巧

* 站内搜索 keyword site:baidu.com
* 标题搜索 intitle：keyword
* 替代词 理想*义
* 关键词不打散,按照关键词的先后顺序查找网页内容 “keyword”
* 文件类型 keyword filetype:pdf
* 短横（-）：排除你不想要的结果
