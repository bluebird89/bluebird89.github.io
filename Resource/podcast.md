# Podcast

* 爆发
  - 声音场景的丰富：虽然已经进入所谓“短视频”时代，但通勤、散步、工作、睡前。步行或手里做事，眼睛被占据的场景还是相当丰富的；
  - 疫情空闲期的激发：我发现很多新播客是疫情期间大家不得以待在家中，想做一些事情之时的一个选择。因为播客真的是一种门槛和成本比较低的创作方式。
  - 疫情期间的视频会议：ZOOM等视频会议形成，提供了一种比较成熟的远程沟通手段，这为谈话和访谈型的播客进一步降低了创作门槛。
  - 相对而言的净土：视频、公众号、微博类平台大多进入一种泥沙俱下的状态，播客的量级太小，天然地排除大量投机型流量型的媒体，让播客的整体质量看上去还不错
* 坚持
  - 维生素E的主播VE主要认为：
    + 确实对讲的内容有极大的兴趣，做播客是一个督促持续学习这些内容的方式。
    + 有很多实际的收获，通过做播客认识了很多朋友和听众，现在成了主要的朋友圈之一。另外的逻辑水平和讲话的水平也直线提升，这显示在了日常的工作内容上面。
    + 这个播客也给了我很多的自信，知道自己做的东西有人盼着听是一件非常有成就的事情
  - Music ONLY Podcast的主播Sarah说：
    + 其中一位主播是非常有水平的人，音乐方面的洞察比现在市面上的乐评人深太多，介于现在电台dj的身份很多话不能说很多歌不能放。因此我希望能够让更多人听到他对音乐的真正洞察。
    + 另一方面也是自己对音乐的热爱没有减退过，每次听到好听的令人想要拍大腿的音乐就真的很希望让更多人听到。
  - 尤比克电台的主播赵木子说（和VE很像）：
    + 我工作外的社交圈比较窄，这个能帮助我认识更多人。
    + 通过这个播客，能强迫自己阅读量和新信息获取量变大。这两个简单来说就是这事儿对我足够有趣和有成就感。
    + 我们两个主播大学是学电视的，虽然现在的职业与电视基本无关，但有一种老编导不能开天窗的“责任感”，这东西还挺根植于思维模式里面的，于是就基本上几乎一期都没停更地坚持了下来。
  - 打个电话给你的主播Lphie说（第一点我也有很多感受）：
    + 我们开始做大概三四个月以后，有听众给我们来信说，我们的电台陪伴他们度过了一些低潮的时期，听我们聊天就像多了两个朋友一样，絮絮叨叨但是很亲切。
    + 我觉得可以带给人陪伴的感觉是最大的动力吧。另一个原因是，我们的听众有非常多来自港澳台、海外留学生和学中文的外国人。他们会常常跟我们互动对于同一个话题的不同看法。这个podcast如果可以作为一个桥梁，去听到不同的声音、建立跨地域之间的关系，是我很期待的。
  - 花市电台的主播乐先生说（主要在探索新的可能）：
    + 想试试在一个非商业性的、有主导权的事情上，最后能做成什么样。也觉得和现在的社会环境格格不入的，应该不止是只有我们这一小部分人。凑一凑，看以后还能玩出什么更大更好玩的事。往糙了说的话，设备买了不能浪费是不是，至少得对得起设备
  - 野点电台（已经更新近170期）的ziondandada认为：因为保持更新是做之前就想好的目标，野点的节目做起来不难，四个人轮流出也算轻松（可能没说的这么轻松）

  - 我自己从2015年底开始制作“优斯迪吧式”的聊天型播客，到2016年中旬正式开始做单口知识型播客，到现在也快5年了。在很多人看起来，我已经成了播客界的“前辈”。
