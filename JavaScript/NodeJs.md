# 安装

```
# ubuntu
sudo git clone https://github.com/nodejs/node.git
sudo chmod -R 755 node
cd node
sudo ./configure
sudo make
sudo make install

curl http://npmjs.org/install.sh | sh

# Mac
brew install nvm
nvm install 4.4.5
nvm use 4.4.5

brew install node
```
