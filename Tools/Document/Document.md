## [jgm/pandoc](https://github.com/jgm/pandoc)

Universal markup converter http://johnmacfarlane.net/pandoc
John MacFarlane开发的标记语言转换工具，可实现不同标记语言间的格式转换.

* 将Markdown转化为Word，然后统计字数

```
pandoc WEB.md -o web.docx
```
