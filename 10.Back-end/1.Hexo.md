# 搭建

```
brew install git
brew install node
npm install hexo-cli -g
cd filename
hexo init
hexo g(enerate)
hexo s(erver)
npm install hexo-deployer-git --save
npm install(hexo 会找不到)
修改_config.yml,添加仓库
type: git
repo: git@github.com:bluebird89/bluebird89.github.io.git
branch: hexo
hexo deploy
```
