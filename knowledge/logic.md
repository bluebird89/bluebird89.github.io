  - 每个人都或多或少地活在一个或小或大的「信息回音壁」里，没有谁的观念、谁的想法是100%绝对正确的。

## 维特根斯坦

### 逻辑哲学论

* 这本书也许只有那些自己已经思考过在此书中所表述的思想或者类似的思想的人，才能理解。
* 凡是能够说的事情，都能够说清楚，凡是不能说的事情，就应该保持沉默。
* 语言是关于经验的描述，但并非所有思想都是关于经验的，因此语言可以表达的东西变得十分有限。同理，哲学研究的内容也不属于经验，但人们在思想和表达哲学问题时，只能用语言来描述，因而导致这些描述变得无意义。
* 一切哲学问题都是对语言的误用”，是因为语言被误用在了它无法描述的事物上，以致前人讨论的 “哲学问题”都是无意义的
* 致力于给思想划界，也就是在澄清什么东西是可言说的，在找到可言说的事物的边界之后（可说的要说清楚），不可言说的东西自然就在边界之外显现（不可言说的要保持沉默），以此来避免因人们硬去言说不可言说的东西而产生所谓的“哲学问题”。
* 认为可以说出来的事物只是最小的部分，而那些不可言说的的部分（包括哲学）才是最重要的，因此他只是否认了人们以往通过语言来表达哲学的方式。
* 例子
  - 功利主义 utilitarianism – 语义模糊
    + 中文里的功利多有功名利禄的意思
    + 在十八世纪末才被边沁和穆勒提出成为道德哲学的一个理论，出现的时间最早可以追溯到昔兰尼的阿瑞斯提普斯和伊壁鸠鲁提出的享乐主义，即主张人终其一生追求的不过是快乐。同样的，功利主义虽有主张各不相同的众多流派，但无论哪种流派都是基于对快乐的追求，因此功利主义可以理解为享乐主义的进阶版，即致力于把享乐主义追求的快乐量化。功利主义主张快乐是善，每个人都追求快乐，而痛苦是不好的，要避免痛苦的产生。如果一个行为有助于增加快乐，也就是增加善，这个行为就是正确的，反之如果这个行为会导致痛苦的增加，那就是错误的。因此，想要一个社会达到最大的快乐总值（最大的善），就要把与此相关的所有个体的快乐和痛苦纳入考量，以此来判断采用何种策略能让整体快乐值最大化，达到最大效益，因此功利主义最恰当的叫法是“效用主义”。
    + 功利主义是从个人角度出发的，注重人的价值和个体之间的平等，即每个人都被视作一个人，不能被视作多余或少于一个人。正因如此，把功利主义概括为“为了整体效益而牺牲少数人的功利行为”的说法是对其极大的误解，甚至可以说， “功利主义”与“功利”在中文里的含义几乎没有任何联系。但由于功利主义更注重结果而非动机，与中文的“功利”不谋而合，导致功利主义在中文里和“功利”混为一谈。作为道德哲学和政治哲学的重要组成部分，功利主义在中文里被如此不准确地理解是非常遗憾和危险的。
    + 对功利主义的误用甚至比不了解功利主义还要可怕，因为它极大地封锁了中文语境里人们对于功利主义的思考和应用，也封锁了诸多对社会有利的政策的可能性。这样的矛盾正是维特根斯坦所说的由于“对语言的误用”而产生的无意义的问题，而这本可以通过用“效用主义”替代功利主义，用语意澄清来避免一部分的。
  - 女权主义 feminism – 望文生义
    + 讨论是建立在参与讨论的人都明确在讨论内容范围的情况下进行的对话，但如今很多关于女权主义的所谓的“讨论”都是建立在不了解或者误解“女权主义”的基础上的，导致围绕这些话题的对话像是搭错了的电话线，往哪说都是无意义的。因此也有必要对“女权主义”的含义做一些澄清。
    + “权”代表的是“权利”（right）而不是“权力”（power）。权利主要指代人权，即作为人类应该享有的权利，例如生命权和自由权，是值得被争取的。而权力多指对他人他物的控制和影响，是需要被警惕的。生活在十八世纪欧洲的女权主义倡导者正是因为察觉到了女性作为人类应该享有的权利在社会中的缺席，才发起了关于女权主义的运动，以此来为女性争取作为人的权利。这种主张在现代化的文明社会，是无论作为何种性别都可以理解的，因为人们已经建立起了“所有人类都需要人权”的共识。女权主义同废奴主义一样，是人类社会的觉醒和进步，因此在概念上来说，反对女性权利也是反对人权。然而女权主义在中文语境里的应用还是导致一些没有必要的矛盾，其中一个原因就是“女权”在中文里容易被望文生义。
    + 从字面上理解的话，很容把它放在与“父权”（patriarchy）同样的层面上相对。诚然，女权主义运动是对父权社会的一种否认与反抗，但必须注意的是，父权中的“权”指代的是权力，主要强调男性在社会中的支配地位，和以男性为权力中心的运作模式，这与女权主义中强调的“权利”完全不同，更不能放在同一个层面来对比。父权社会对于女性的压迫与歧视在当下仍然是显而易见的，所以以争取女性权利为目标的女权主义运动是一定会对父权社会造成冲击甚至瓦解的，但这并不代表女权主义者要用一种与“父权”对立的，以女性作为权力中心和支配地位的社会运作模式来代替父权，从而反过来压迫与歧视男性。恰恰相反，女权主义追求的是性别平权和反歧视，因此女权主义运动也是平权运动的一种，只是女权主义更多是从女性经验的角度出发的。这也是为什么女权主义在中文里为了避免歧义应叫作“女性主义”。
    + 女权主义运动要做的是教化人们找到文明，使用理性，并且明白父权社会造成的悲剧不是能单纯地甩给某个性别的人群的，要改变这种现状是需要所有性别一起努力的，而这与当下中文互联网上越来越严重的男女性别对立可谓背道而驰。
  - 公共知识分子/理中客 – 污名化
    + 具有专业知识背景的他们正如波斯纳在《公共知识分子-衰落之研究》中所说的：“以社会公众可以接近的方式表达自己，并且其表达聚焦于有关或涉及政治或意识形态色调的社会公众关注之问题。”在那个时候，公知是受人尊敬的，因为人们渴望知识，而公知愿意传播知识。到了二十一世纪初期，虽然知识分子不再以群体的形式存在了，但社会大众仍然关注他们的看法。
    + 当人们用站队代替思考时，他们不再需要知识，也不再需要知识分子提供的观点，毕竟党同伐异只需要粗暴割席。因此，公共知识分子的消失也代表着理性的衰落和反智的崛起。
    + 在“公知”这个词已经完全变味之后，那些真正的“公共知识分子”又该如何自处呢？他们没有办法自处，因为没有别的词汇可以代替“公共知识分子”。在“公知”被污名化后，那些仍以传播知识和观点为己任的知识分子们已经找不到别的词语来称呼自己，于是公共知识分子原本所代表的概念在中文互联网中消失了，而人们并不在乎公知们在教化民众方面扮演的重要角色。把公共知识分子取而代之的是被误用的“公知”，于是劣币再一次驱逐了良币。
    + 公共知识分子的缺席其中一个根源是理性的缺席，而理性的缺席是由于对理性的误用导致的。这里要强调，理性指的是人运用理智的能力，可以看作是人的基石。换句话说，能意识到自我的人类都是有理性的。理性本是个无可指摘的词，却常常因为与感性对立而产生矛盾。
    + 感性指的是人对一切事物的直接感觉而产生的感受，但这种感受是经过意识的反馈才得以实现的因此感性（包括人的共情能力）都属于在理性的一部分。
    + 理中客原意理性，中立，客观，本身并无黑点，但是总有一些打着“理性分析”旗号的文章在人类运用共情能力时假装清醒，并对人们的情绪发出质疑。
    + 在任何时候理性思考都不应该被摒弃，因为理性与人类所有的悲欢喜乐并不冲突。理性告诉人们，要把自己当作人，也要把别人当作人，理性的缺席就是人性的缺席。
  - 房间里的大象 – 描述不可说之物
    + 人们正在努力用各种东西把房间填满，并尝试描述房间里面除了大象之外的所有细节，希望可以因此试探到大象的边界，并把握到它的轮廓。在中文语境里，人们恰恰需要误用语言去打破沉默，并让矛盾得以显现。
    + 规定语言的使用方法是当权者权威的体现。当描述房间里的东西也变得困难，而人们也不被允许发明另外一种语言去描述有关大象的一切信息时，他们只能用当权者规定的语言去描述这种语言范围之外的东西，于是还是陷入了维特根斯坦所说的“用语言描述不可描述之物”的陷阱中，又一次对语言进行了误用。但这种误用与前文提到的误用是完全不同的，因为误用语言的人是自省的，或者可以说，故意误用中文是需要表达的人在中文互联网的生存之道。
    + 维特根斯坦说“语言是思想的界限”，是因为人的一切思想都由语言构成，人们无法描述思想之外的东西时，也就无法思考语言之外的内容。当母语不再是家园，当它代表了权威，并且被用来筑起高墙时，使用母语的人只能从语言的深处为思想寻找家园。

## 实例

```
“某某著名大学实验室通过基因测序发现，非典病毒包含对黄色人种极具攻击性的变异细胞。在北纬2X度，东经12X度的地方，这种细胞特别容易裂变繁殖。而且在体内黄色素居多的黄色人种的肺部非常容易存活，但在白色素居多的白色人种和黑色素居多的黑色人种的肺部却无法存活。
所以，这个病毒极有可能是专门研究出来攻击某个地理区域的黄色人种的。这一区域正好与中国大部分地区重合。美国一直在秘密研究基因武器，而且美国敌视我们，所以，非典极有可能是美国对我们发动的生化战争。”
```

* 这段文字先用一连串貌似高端的术语，给非专业人士一个杀威棒，让他们觉得文章的论据非常专业，所以，十分有力。那段文字中所谓“黄色素”、“白色素”、“黑色素”就是诉诸于人的直觉而虚构的专业术语。“白种人身上白色素多，黄种人身上黄色素多，黑种人身上黑色素多”的说法，对于大多数非专业人士来说，听起来非常专业，而且符合直觉。在这种“专业性”所营造的权威感面前，大多数人都会相信。
* 以这些“专业论据”作为逻辑起点，文章用流畅平实、简单易懂的叙述，展开符合逻辑、环环相扣的论证。
* 得出最终结论——美国的阴谋——之前，另外两个关键“论据”被抛了出来：
  - 美国一直在秘密研究基因武器；
  - 美国一直敌视中国。
* 由于是“秘密”，美国是否真的在开展这项研究，大多数人根本无法查证，也懒得查证，甚至想都没想过是否需要查证，就稀里糊涂相信了，并把它当证据。少数人即使心存怀疑，也无法对其进行证伪，因为它是“秘密”。
* “美国一直敌视中国”，这个能不能成为证据？我相信，很多中国人会把它当做证据。如果是基于严谨的科学精神，它不能成为证据，因为它是观点，不是事实。两者的区别在于，观点是无法证明的，而事实是可以证明的。
* “坏事发生是因为有人在暗中搞鬼”非常符合直觉。由于病毒“未知的来源”引起的恐慌需要安抚，“美国秘密阴谋”这一答案，既能满足直觉，又能让大家恢复安定生活所需要的确定感。思维惰性让很多人相信了它。
